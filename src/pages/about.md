流云栖木是一个面向设计、代码与系统思考的个人博客。它从 [AstroPaper](https://github.com/satnaing/astro-paper) 出发，保留静态性能、SEO、RSS、搜索与文章归档，再逐步发展出更像纸面排版的视觉和内容组件。

这里的目标不是把界面做重，而是让每一层都更准确：字体有角色，颜色有克制，组件有边界，文章页面适合长时间阅读。半卷纸痕，几行代码；桨声入夜，灯影成篇。

## 原则

- 文字优先，视觉服务阅读。
- Astro 负责内容与静态输出，React 只用于值得交互的小岛。
- Tailwind CSS 与 shadcn/ui 提供一致的设计令牌和组件基础。
- 中文、英文、代码、旁注与日文题签各自使用明确的字体角色，不依赖系统默认字体作为主要呈现。

## 技术栈

Astro 6, Tailwind CSS 4, React 19, shadcn/ui, Pagefind, Shiki, RSS, sitemap, dynamic OG images.

## 后续

后续可以继续细化文章封面、目录、评论、订阅、项目页、系列文章、阅读进度、主题预设和更多内容组件。
